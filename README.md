# ArijkxPixels Beta Website

**Arijkx PIXELS** - A pixel art portfolio website with gamification features

This website is currently in **BETA** and is not a final version.

[Visit the beta site](https://arijkx.github.io/arijkxpixels/)

## Features

### Level System
- **Automatic XP Gain**: Earn +5 XP per second while browsing the website
- **Level Progression**: Level up by accumulating experience points
- **Gold Currency**: Earn gold by completing quests and use it to unlock rewards
- **Local Storage**: All progress is saved in your browser's local storage

### Quest System
- **Interactive Quests**: Complete various quests by interacting with different parts of the website
- **Quest Rewards**: Earn bonus XP and Gold for completing achievements
- **Quest Tracking**: Track your progress on multiple quests simultaneously
- **Quest Categories**: Different types of quests including exploration, interaction, and achievement-based

### Rewards System
- **Unlockable Wallpapers**: Purchase high-quality pixel art wallpapers with earned Gold
- **Multiple Resolutions**: Wallpapers available in various resolutions (1280x720, 2560x1440, 3840x2160, ultra-wide formats)
- **Preview System**: Preview wallpapers before purchasing
- **Download System**: Download unlocked wallpapers directly to your device

### Visual Features
- **Parallax Header**: Multi-layer parallax scrolling background with animated layers
- **Pixel Art Design**: Retro pixel art aesthetic throughout the website
- **Responsive Layout**: Adaptive design for different screen sizes
- **Animated Elements**: GIF animations and sprite-based animations

### Content Sections
- **About**: Learn more about the artist Arijkx
- **Game Assets**: Browse retro pixel art game assets
- **Game Development**: View game development projects
- **News Feed**: Latest updates and news from Arijkx (Mastodon integration)
- **Services**: Overview of available services
- **Pixel Teachers**: Discover masters of pixel art
- **Partners**: Trusted partners and affiliates

### Games
- **SpaceFleet**: Interactive space shooter game (coming soon)
- **Game Projects**: Showcase of current and upcoming game development projects

### Social Integration
- **Social Media Links**: Connect via Twitter, Instagram, Twitch, YouTube, Discord, and Ko-fi
- **Mastodon Feed**: Integrated social media feed
- **Support Links**: Direct links to support the artist on Ko-fi

### Technical Features
- **Component-Based Architecture**: Modular header, footer, and scroll-to-top components
- **Dynamic Content Loading**: JavaScript-based content loading system
- **Local Storage Management**: Persistent data storage for game systems
- **Cookie-Free**: No cookies, no tracking - privacy-focused design

## Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with pixel art aesthetic
- **JavaScript (ES6+)** - Interactive features and game systems
- **Local Storage API** - Data persistence
- **Canvas API** - Game rendering
- **Mastodon API** - Social feed integration
